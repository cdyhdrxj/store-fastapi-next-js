1. Реализовать backend на одном из фрейморков (laravel, yii, flask, fastapi, express) для системы интернет-магазина. API должно предоставлять следущие функции:
    - Работа с товарами: добавление, удаление, получение списка (с пагинацией и возможностью поиска), изменение.
    - Работа с категориями товаров: добавление, удаление, получение списка, изменение. Один товар может быть привязан только к одной категории.
    - Список изображений для товара: добавление, удаление, получение списка, изменение.
    - Бренды товара: добавление, удаление, получение списка, изменение. Товар может быть связан только с одним брендом.
    - Использовать: orm, migration, swagger для демонстрации.

2. В реализованное на этапе 1 приложении добавить пользователей и роли (администратор (все права), менеджер (добавлять, изменять, удалять объекты в системе) и обычный пользователь). Авторизацию/аутентификацию реализовать с помощью jwt - токенов.

3. Добавить возможность сделать авторизиваронным пользователем покупку (без оплаты).

4. Реализовать интерфейс админки для интернет магазина (опираясь на backend, разработанный на первых этапах): работа с товарами, категориями, брендами и изображениями для товаров. Использовать фрейморк: react или vue и библиотеки mui или vuetify.

5. Добавить в интерфейс роли: администратор может все (в том числе удалять, добавлять пользователей).

6. Реализовать интерфейс покупки товара: список товаров (изображение, название и описание товара).

7. Добавить следующую возможность: когда пользователь покупает товар, то у менеджера всплывает сообщение об этом (без перезагрузки страниц). Использовать websocket протокол.
